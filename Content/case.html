Case Commands

Vu
Lowercase Line
VU
Uppercase Line
g~~
Invert Case of Line
vEU
Uppercase Word
vE~
Invert Case of Word
ggguG
Lowercase Entire Buffer
gggUG
Uppercase Entire Buffer
:set ignorecase
Ignore case in all searches
:set smartcase
Ignore case in all searches unless an uppercase letter is used
:%s/\<./\u&/g
Uppercase the first letter of each word
:%s/\<./\l&/g
Lowercase the first letter of each word
:%s/\<.*/\u&/g
Uppercase the first letter of each Line
:%s/\<.*/\l&/g
Lowercase the first letter of each Line
