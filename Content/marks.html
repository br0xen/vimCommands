Marks

Marks allow you to jump to a specific position. Each file can have marks labelled a-z. In addition, there is a set of global marks identified by uppercase letters (A-Z).

So, for example, if you want to be able to easily jump to your .vimrc file, open it up and set a capital letter mark in it (i.e. 'mA'), now whenever you hit the jump to mark command and specify 'A' as the mark to jump to, it will take you to your .vimrc file.

mk
Marks the current position as k
'k
Move the cursor to Mark k (first non-blank character in line)
`k
Move the cursor to Mark k (line and column)
d'k
Delete from the current line to line of mark k
d`k
Delete from the current cursor position to position of mark k
Delete from the current line to line of mark k
c'k
Change all text from current line to mark k
y`k
Yank text from cursor to mark k
:marks
List all current marks
:marks aB
List marks a and B

Special Marks

Vim has some special marks that are automatically set, here are a few

`.
Jump to position where last change occurred in current buffer
`"
Jump to position where last change occurred in current buffer
`0
Jump to position in last file edited (when Vim last exited)
`1
Like `0, but the previous file (also `2, etc.)
''
Jump back to line in current buffer that you just jumped from
``
Jump back to position in current buffer that you just jumped from
` [ or ` ]
Jump to beginning/end of previously changed or yanked text
` > or ` <
Jump to beginning/end of last visual selection

See the full list at :help '[ (and following)
