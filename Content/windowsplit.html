Window Splitting

:e filename
Edit filename in current window
:split filename
Split the window and open filename
Ctrl-w Up
Move the cursor to the top window
Ctrl-w Ctrl-w
Move the cursor to the next window
Ctrl-w_
Maximize the current window
Ctrl-w=
Gives the same size to all windows
10 ctrl-w+
Add 10 lines to current window
:vsplit filename
Split window vertically and open filename
:sview filename
Split window and open filename in readonly mode
:hide
Close current window
:nly
Close all windows except the current one
:b 2
Open buffer 2 in this window
